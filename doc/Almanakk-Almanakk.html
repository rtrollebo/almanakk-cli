data CellestialPhaseEvent Source #

Cellestial phase event type parameter for phase functions

New: beginning of the phase cycle, and the first phase event, corresponding to the event when the shadow cast by the primary celestial body onto the satellite disk is at an minimum FirstQuarter: the phase event between new and full when the shadow of the satelitte disk is 50% and Full: the phase event when the shadow of the satelitte disk is 100% LastQuarter: the phase event at the the end of the phase cycle, between full and new, when the shadow of the satelitte disk is 50%
