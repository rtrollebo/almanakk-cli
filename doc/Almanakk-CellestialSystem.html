data CelestialBody Source #

Celestial body

This is the base type for all celestial bodies. To create a celestial body, either use the CelestialBody constructor, or the CelestialBodySolarSystem constructor for solar system planets.

CelestialBodyType: Star, Planet, SubPlanet String: name of the celestial body CelestialBodySolarSystem: Solar system planet (Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune)
